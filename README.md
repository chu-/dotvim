dotvim
======

My vim files

**deloyment via git**

    git clone https://github.com/chu-/dotvim.git ~/.vim

    ln -s ~/.vim/vimrc ~/.vimrc

    cd ~/.vim

    git submodule init

    git submodule update

**deployment via vundle**
    
    git clone https://github.com/chu-/dotvim.git ~/.vim

    ln -s ~/.vim/vimrc ~/.vimrc

    run `:BundleInstall` in vim or gvim
